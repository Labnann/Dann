Dann(inputs,outputs)

  Dann.addHiddenLayer(neuronCount, activation_function)
    adds one layer.

  Dann.makeWeights()
    creates the weights.
    Should be called after the creation layers.
    
  Dann.activation(layer_index,activation_function)
    set the activation of a specific layer.

  Dann.setActivations(activation_function)
    set all the activation in the NeuralNetwork

  Dann.feedForward(inputs[])
    feed forward the neural network.

  Dann.backpropagate(inputs[],targets[])
    backpropagate the neural network.

  Dann.log()
    console.logs info on the neural network.

  Dann.save()
    gives you a string of all the weight matrices, copy it!

  Dann.load(stringedMatrix)
    you can load weights by giving this function the previously acquired matrix string given by "Dann.save()".
